## 1. 分区标识规范

> 全局唯一定义，其他文档引用此表。

| 分区 | Emoji | 文字 | 判定规则 | 说明 |
|------|-------|------|----------|------|
| 历史热区 | 🔥 | [热] | 评分排名前 hotPercent | 历史高频使用，优先展示 |
| 新兴热区 | 🔥 | [新兴热] | 48小时内使用过 + 保留席位 | 最近频繁使用 |
| 常温区 | 🌡️ | [常温] | 中间 warmPercent | 中频使用 |
| 冷区 | ❄️ | [冷] | 剩余内容 | 低频使用 |
| 导入 | 📥 | [导入] | isImported=true | 从外部知识库导入 |

**注意**：分区标识基于 `partition_config` 中的相对排名，不使用硬编码阈值（评审 P0 改进）。

## 4. 参数控制

### 4.1 索引展示参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--hot-count` | 热门索引展示个数 | 5 |
| `--depth` | 索引层级深度（1-10） | 4 |
| `--partition` | 分区过滤：hot/warm/cold/emerging/all | all |
| `--mode` | 展示模式：full/hot/compact/help | full |
| `--emerging` | 只展示新兴热区内容 | false |

### 4.2 Relations 展示参数

| 参数 | 说明 | 默认值 |
|------|------|--------|
| `--hot-count` | 热门知识展示个数 | 5 |
| `--keyword-limit` | 关键词展示数量限制 | 20 |
| `--show` | 展示内容类型：relations/keywords/all | all |
| `--partition` | 分区过滤 | all |
| `--mode` | 展示模式：full/compact | full |
